# malek-sae-php-oracle
Projet académique PHP avec base de données Oracle (OCI8)
## Technologies
- PHP
- Oracle Database
- OCI8
- HTML / CSS

## Prérequis
- PHP avec extension OCI8
- Oracle Instant Client
- Base Oracle configurée

## Remarque
Le projet nécessite une configuration Oracle locale pour fonctionner.
Les identifiants de connexion ne sont pas inclus.
