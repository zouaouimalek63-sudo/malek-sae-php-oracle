# Projet PHP / Oracle (OCI8)

Projet académique développé en PHP avec une base de données Oracle.

## Technologies
- PHP
- Oracle Database (OCI8)
- HTML / CSS

## Installation / Lancement
1. Placer le projet dans le dossier htdocs (XAMPP)
2. Démarrer Apache
3. Configurer la connexion Oracle (fichier de configuration non inclus)

## Remarques
- Les identifiants et paramètres de connexion ne sont pas inclus dans ce dépôt.
- Certaines fonctionnalités peuvent nécessiter une base Oracle déjà initialisée.
